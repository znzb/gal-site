# 后台管理系统使用说明

## 访问地址

- **前台网站**: https://galzb.cc.cd
- **后台管理**: https://galzb.cc.cd/admin.html

## 默认账号

- 用户名: `admin`
- 密码: `admin123`

## 功能模块

### 1. 控制台 (Dashboard)
- 查看网站统计数据
- 最近添加的游戏列表

### 2. 游戏管理 (Games)
- 查看所有游戏
- 添加新游戏
- 编辑游戏信息
- 删除游戏
- 按分类筛选
- 搜索游戏

### 3. 分类管理 (Categories)
- 查看所有分类
- 添加新分类
- 编辑分类
- 删除分类

### 4. 轮播图管理 (Banners)
- 查看所有轮播图
- 添加新轮播图
- 编辑轮播图
- 删除轮播图
- 设置排序

### 5. 公告管理 (Announcements)
- 查看所有公告
- 添加新公告
- 编辑公告
- 删除公告
- 设置置顶
- 设置显示/隐藏

### 6. 补档管理 (Patch Requests)
- 查看所有补档申请
- 审核补档申请（批准/拒绝）
- 标记补档完成

### 7. 搜索统计 (Search Logs)
- 查看热门搜索关键词
- 查看搜索次数统计

## 部署说明

### 推送到 GitHub
```bash
git add .
git commit -m "添加后台管理系统"
git push origin main
```

### 前端部署 (Vercel)
代码推送后，Vercel会自动重新部署。

### 后端部署 (Render)
代码推送后，Render会自动重新部署。

## 安全建议

1. 首次登录后请立即修改默认密码
2. 定期更换管理员密码
3. 不要将管理员账号共享给他人

## 常见问题

### 忘记密码怎么办？
如果忘记了管理员密码，需要直接在数据库中修改：
```javascript
// 连接到 MongoDB Atlas 数据库
// 找到 admin 用户，使用 bcrypt 重新加密密码
```

### 如何添加新的管理员？
需要直接在数据库中创建新用户，并使用 bcrypt 加密密码。

### 数据怎么备份？
建议定期在 MongoDB Atlas 中进行数据备份。
